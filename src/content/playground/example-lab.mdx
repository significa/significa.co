We kept rebuilding the same accessible primitives across client projects. So we extracted them into a library.

## What it is

A collection of unstyled, fully accessible UI components for Svelte. Dialogs, dropdowns, tabs, toggles — the boring stuff that's surprisingly hard to get right.

## Why unstyled?

Styled component libraries are great until your design doesn't match. We wanted primitives that handle all the hard parts — keyboard navigation, focus management, ARIA attributes — while leaving visuals entirely to you.

## Usage

Install it, import a component, style it however you want. Every component follows the WAI-ARIA authoring practices and handles edge cases like focus trapping, scroll locking, and screen reader announcements.

Built from the patterns we refined over dozens of client projects. Battle-tested in production.
