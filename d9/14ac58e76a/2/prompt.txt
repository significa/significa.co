**Adiciona a secção: Mosaic Icon System**

Icons são grids 7×7 de elipses CSS renderizadas como "eggs".

Ficheiros:
- src/components/mosaic-icon.astro — renderer
- src/lib/mosaic-icons.ts — definições de icons

Formato de cada icon:
```ts
interface MosaicIconDef {
  cols: number           // normalmente 7
  idle: MosaicLevel[]   // flat array, length = cols × rows
  hover?: MosaicLevel[] // opcional, mesmo tamanho
}

type MosaicLevel = 0 | 1 | 2 | 3 | 4
// 0 = invisível | 1 = 27.5% | 2 = 57.5% | 3 = 87.5% | 4 = 100% da célula
```

Array é row-major: primeiros `cols` valores = primeira linha, left-to-right, top-to-bottom.

Hover mechanism: CSS custom property no pai.
- Pai define --mosaic-state: 0 (idle) ou --mosaic-state: 1 (hover) via :hover
- O MosaicIcon interpola automaticamente via calc()
- Zero JS. Zero React. Zero data attributes.

Tamanho: atualmente hardcoded a 2px/célula (14px para grid 7×7).
TODO (Session 2): adicionar prop size para suportar 14, 16, 20, 24px.

Para adicionar um icon novo: seguir o padrão exato do arrow existente em mosaic-icons.ts.
Nunca criar SVG icons para elementos que pertencem ao sistema mosaic.

Cor: os icons herdam cor do pai via currentColor. Sem prop de cor.
