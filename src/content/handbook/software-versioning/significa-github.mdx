At Significa, we host most of our repositories — both public and private — on GitHub. Pull requests are our primary method for managing workflow, while we utilise Linear for all client-based projects. For open-source initiatives, we leverage GitHub issues to keep contributions transparent and accessible for everyone interested.

Contributing to our open-source repositories is simple with your own GitHub account, ensuring you receive credit for your work. Learn all the details about this in Open source.

If you're starting your Engineer onboarding, remember to set up your Git using SSH and enable commit signing.
